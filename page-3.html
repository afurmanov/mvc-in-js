Controller

Can:

  • this.model.method(); //call public model inteface
  • Ember.set(model, 'property', value); //change model property
  • this.view.method(); //ask view to do something
  • this.view.on('ELEMENT.CHOSEN', function() {...// listen for view events
  • this.model.on('PROPERTY.CHANGED', function() {...//listen for model events
  • this.model.reloadFromServer.done( function(data) {...//listen for model async events
  • watchProperty : function() {...}.observes('model.property');//observes model property

Cannot:

  • $('.property').hide(); //manipulate DOM directly
  • this.view.$property.hide(); //manipulate DOM using view internals
  • this.model.property += 1; //change model bypassing model interface
