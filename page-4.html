View

Can:

  • this.model.getProperty(); //reading model properties
  • $('.element').hide(); //rendering
  • $('.element').on('click', ... //listen for user input
  • NB.EventHub.emit('BUTTON.CLICKED', function() {... //emitting events
  • function() {...}.observes(model, 'property'); //listen for model property changes

Cannot:

  • this.model.updateProperty(); //update model
  • Ember.set(model, 'property', value); //update model
  • controller.method() //use controller directly
